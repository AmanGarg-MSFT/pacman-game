# Pac-Man Game

A classic Pac-Man game built with HTML5 Canvas, CSS, and JavaScript.

## Features

- Classic Pac-Man gameplay
- Grid-based movement
- Intelligent ghost AI that chases the player
- Score tracking and lives system
- Ghostly visual effects
- Win/lose conditions

## How to Play

1. Use arrow keys to move Pac-Man around the maze
2. Eat all the yellow dots to win
3. Avoid the red ghost - if it touches you, you lose a life
4. You start with 3 lives
5. Collect all dots to win the level!

## Game Controls

- **Arrow Keys**: Move Pac-Man (Up, Down, Left, Right)
- **F5**: Restart the game
